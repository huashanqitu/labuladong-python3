单调栈总结  
单调递增栈：从 栈底 到 栈顶 递增，栈顶大  
单调递减栈：从 栈底 到 栈顶 递减，栈顶小  

1：什么时候使用单调栈？  
通常是一维数组，要寻找任一元素右边（左边）第一个比自己大（小）的元素，且要求 O(n) 的时间复杂度  

2：模板套路  

1): 当前项向右找第一个比自己大的位置 —— 从右向左维护一个单调递减栈  
```
def nextGreaterElement_01(nums: list):
    length = len(nums)
    res, stack = [-1] * length, []

    for i in range(length - 1, -1, -1):
        while stack and stack[-1] <= nums[i]:
            stack.pop()
        if stack:
            res[i] = stack[-1]
        stack.append(nums[i])

    return res
```   
或者 当前项向右找第一个比自己大的位置 —— 从左向右维护一个单调递减栈
```
def nextGreaterElement_011(nums: list):
    length = len(nums)
    res, stack = [-1] * length, []

    for i in range(length):
        while stack and nums[stack[-1]] < nums[i]:
            idx = stack.pop()
            res[idx] = nums[i]
        stack.append(i)

    return res
```

### 下一个更大元素 I  单调栈
法一
```
# 当前项向右找第一个比自己大的位置 —— 从右向左维护一个单调递减栈
class Solution2:
    def nextGreaterElement(self, nums1, nums2):
        dic, stack = {}, []

        for i in range(len(nums2) - 1, -1, -1):
            while stack and stack[-1] <= nums2[i]:
                stack.pop()
            if stack: dic[nums2[i]] = stack[-1]
            stack.append(nums2[i])

        return [dic.get(x, -1) for x in nums1]

```

法二
```
# 从左到右维护单调递减栈， 找元素右侧区域，第一个比自己大的位置
class Solution3:
```

### 力扣第 1118 题「一月有多少天」

```

class Solution:
    def dailyTemperatures(self, T):
        res = {}
        # 这里放元素索引 而不是元素
        s = []
        for i in range(len(T)-1, -1, -1):
            while s and T[s[-1]] <= T[i]:
                s.pop()
            # 得到索引间距
            if s:
                res[T[i]] = s[-1] - i
            else:
                res[T[i]] = 0

            # 将索引入栈 而不是元素
            s.append(i)
        return [res.get(j) for j in T]

s = Solution()
T = [73,74,75,71,69,76]
res = s.dailyTemperatures(T)
print(res)

```

### 下一个更大元素 II 循环数组
```
class Solution:
    # 我们可以不用构造新数组，而是利用循环数组的技巧来模拟数组长度翻倍的效果
    def nextGreaterElements(self, nums: List[int]) -> List[int]:
        n = len(nums)
        res = [None]*n
        s = []
        # 假装这个数组长度翻倍了
        for i in range(2*n-1, -1, -1):
            # 索引要求模 其他的和模板一样
            while s and s[-1] <= nums[i%n]:
                s.pop()
            if s:
                res[i%n] = s[-1]
            else:
                res[i%n] = -1
            s.append(nums[i%n])
        return res[:n]
```
