### 区间调度
```
def intervalSchedule(intvs):
    if len(intvs) == 0:
        return 0
    intvs.sort(key=lambda x:x[1])

    count = 1

    x_end = intvs[0][1]
    for interval in intvs:
        start = interval[0]
        if start >= x_end:
            count += 1
            x_end = interval[1]
    return count
```

### 无重叠区间

class Solution:
    def eraseOverlapIntervals(self, intervals: List[List[int]]) -> int:
