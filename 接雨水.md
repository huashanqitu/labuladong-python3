### 42. 接雨水
暴力解法

```
class Solution:
    def trap(self, height: List[int]) -> int:
        n = len(height)
        res = 0
        for i in range(1,n-1):
            l_max = 0
            r_max = 0
            # 找右边最高的柱子
            for j in range(i,n):
                r_max = max(r_max, height[j])
            # 找左边最高的柱子
            for j in range(i,-1,-1):
                l_max = max(l_max, height[j])
            # 如果自己就是最高的话
            # l_max == r_max == height[i]
            res += min(l_max, r_max)-height[i])
        return res
```
