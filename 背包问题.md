### 01背包问题
```
def knapsack(W, N, wt, val):
    dp = [[0]*(W+1) for _ in range(N+1)]
    for i in range(1, N+1):
        for w in range(1, W+1):
            if w - wt[i-1] < 0:
                dp[i][w] = dp[i-1][w]
            else:
                dp[i][w] = max(
                    dp[i-1][w],
                    dp[i-1][w-wt[i-1]]+val[i-1]
                    )
    return dp[N][W]
```
