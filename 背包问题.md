### 01背包问题
```
def knapsack(W, N, wt, val):
    # vector 全填入 0，base case 已初始化
    dp = [[0]*(W+1) for _ in range(N+1)]
    for i in range(1, N+1):
        for w in range(1, W+1):
            # 当前背包容量装不下，只能选择不装入背包
            if w - wt[i-1] < 0:
                dp[i][w] = dp[i-1][w]
            # 装入或者不装入背包，择优
            else:
                dp[i][w] = max(
                    dp[i-1][w],
                    dp[i-1][w-wt[i-1]]+val[i-1]
                    )
    return dp[N][W]
```
